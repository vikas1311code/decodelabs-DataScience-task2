# 💳 Credit Card Fraud Detection using Machine Learning

A Machine Learning project that detects fraudulent credit card transactions using classification algorithms. The project includes data preprocessing, model training, evaluation, and performance visualization to identify suspicious transactions accurately.

---

## 📌 Project Overview

Credit card fraud is one of the biggest challenges in digital payments. This project aims to build a machine learning pipeline capable of distinguishing fraudulent transactions from legitimate ones using transaction data.

The workflow includes:

- Data Loading
- Data Preprocessing
- Handling Class Imbalance
- Feature Scaling
- Model Training
- Model Evaluation
- Performance Visualization

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/vikas1311code/decodelabs-DataScience-task2.git
```

Move into the project directory

```bash
cd decodelabs-DataScience-task2
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

## 📈 Dataset

The project uses the popular **Credit Card Fraud Detection Dataset**, containing anonymized transaction features along with a binary target indicating whether a transaction is fraudulent.

Target Values:

- **0 → Legitimate Transaction**
- **1 → Fraudulent Transaction**
